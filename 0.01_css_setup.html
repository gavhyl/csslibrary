Purpose of CSS
  • To give the site a unique look and feel
  • To alter colors, fonts, backgrounds, etc.
  • To implement some effects and animations, although it's not as powerful as jQuery

CSS Statements
  • Declaration: the statement as a whole with curly braces
  • Selector: the name of the part you want to style
  • Property: properties to edit(e.g., with, height, background, etc.)
  • Value: numbers, colors, urls, - values that we assign
